# MCQ Generator

A Streamlit-based app that generates multiple-choice questions (MCQs) from any text using the Groq API. Users can paste text, choose the number of MCQs (1–20), and instantly get well-formatted questions.

---

## Features

- Generate 1–20 MCQs from any text input.
- Clear, well-formatted output.
- User-friendly interface with Streamlit.
- API key safely stored locally in `.env` (never pushed to GitHub).
